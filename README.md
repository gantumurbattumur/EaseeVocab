# EaseeVocab 🎓
Learn English vocabulary with ease using mnemonic techniques, sound-a-like words, and visual stories.

## ✨ Features

- 📚 **Daily Learning**: Get 10 new words every day tailored to your level
- 🧠 **Mnemonic Techniques**: Sound-a-like words help you remember faster
- 🖼️ **Visual Stories**: Generated images create memorable connections
- 🧩 **Interactive Review**: Crossword puzzles and games to reinforce learning
- 📊 **Progress Tracking**: Monitor your streak, XP, and mastered words
- 🌐 **Multi-language**: Learn with Spanish or French translations
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile

## 🚀 Tech Stack

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: FastAPI, Python 3.11
- **Database**: PostgreSQL (Supabase)
- **AI**: Google Gemini API for mnemonic and image generation
- **Deployment**: Vercel (Frontend), Railway (Backend)

## 📖 Getting Started

See [SETUP.md](./SETUP.md) for detailed setup and running instructions.

Quick start:
1. Clone the repository
2. Set up environment variables (see SETUP.md)
3. Install dependencies and run migrations
4. Start frontend and backend servers

## 🎯 How It Works

1. **Learn**: Each day, you receive 10 words matched to your level (A1, A2, B1, B2)
2. **Remember**: Use sound-a-like words and visual stories to create strong memory connections
3. **Review**: Practice with crossword puzzles and interactive games
4. **Track**: Build streaks and monitor your learning progress
